구현할 기능 목록
1. 구독 설정 조회

현재 요금제 정보 (플랜명, 금액, 다음 결제일)
모든 요금제 리스트 (무료 / 프로 / 프리미엄)
현재 플랜 표시
결제 수단 목록 조회 (카드 마스킹 처리)
결제 내역 조회 (플랜명, 결제일, 금액, 상태)

2. 플랜 변경

다른 플랜으로 변경 요청
변경 확인 모달 처리 (취소 / 변경)
변경 완료 후 현재 플랜 업데이트

3. 결제 수단 관리

결제 수단 변경 (빌링키 기반)
새 결제 수단 추가 (빌링키 신규 발급)

4. 구독 해지

해지 요청 (해지 사유 선택 입력)
다음 결제일까지 무료 요금제로 전환 예약
해지 확인 모달 처리 (취소 / 해지)
